# Changelog

All notable changes to this project will be documented in this file.

The format is inspired by Keep a Changelog, and this project uses simple
date-based release notes until public version tags are introduced.

## Unreleased

- Documentation polish for the public GitHub repository.
- Added bilingual README files.
- Added contribution and issue-reporting templates.

## 2026-06-12

Initial clean public release.

### Added

- SwiftUI macOS control app.
- Swift command-line fishing bot.
- Pixel-based detection for:
  - fishing-ready prompt
  - blue hook prompt
  - fishing bar green range
  - fishing bar cursor
  - result prompt
- Balanced `1280x804` window preset.
- Release packaging scripts for app and CLI zip files.

### Notes

- Fishing automation only.
- No auto sell or auto buy bait.
- No Windows or PlayCover support.
