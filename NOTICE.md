# Notice

This is an unofficial macOS fishing helper for the native macOS client of NTE.

It is not affiliated with, endorsed by, or supported by the game developer,
publisher, MaaNTE, or MaaAssistantArknights.

The tool works by reading pixels from the game window and sending normal macOS
keyboard or mouse events. It does not modify game files, game memory, network
traffic, or the game process.

Use at your own risk. Automation may violate a game's terms of service.
